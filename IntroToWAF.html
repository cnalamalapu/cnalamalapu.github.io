Introduction to web app foundations

Overview

In this course, students examine core web technologies that are integral in the creation of web-based applications typically delivered in a browser. The course will cover fundamental web protocols, web application architectures, markup, and scripting languages. A focus will be placed on writing modern, standards-compliant JavaScript and how it is used to interact with HTML and CSS to enable rich user interfaces and communication with other services. Current frameworks, libraries, and tools will also be explored.

  • Recognize HTML, CSS, and JavaScript in web pages.
  • Differentiate between server-side and client-side technologies in web applications.
  • Understand the historical significance of HTML, CSS, and JavaScript in modern web apps.
  • Convert interface mockups into functional prototypes.
  • Explain the role of HTTP and HTTPS in the request/response cycle.
  • Create web pages and apps using valid HTML, CSS, and JavaScript.
  • Debug web apps using browser developer tools.
  • Implement responsive design principles for adaptable layouts.
  • Differentiate between ES5 and ES6+ JavaScript language features.
  • Understand advanced JavaScript concepts like Prototype, OOP, JSON, Namespacing, and Modules.
  • Compare asynchronous content loading techniques, such as AJAX.
  • Utilize HTML5 APIs for enhanced web functionality.
